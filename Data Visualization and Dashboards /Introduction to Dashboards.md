# 🚗 2. What is a Dashboard?

## 🔹 Concept

A dashboard is a visual interface that displays:
- Important metrics
- KPIs
- Charts
- Insights

in one centralized location.

---

## 🔹 Real-World Analogy

A car dashboard displays:
- Speed
- Fuel level
- RPM
- Engine temperature

Similarly, a data dashboard displays:
- Business performance
- Sales
- Revenue
- KPIs
- Trends

---

## 🎯 Interview Point

"A dashboard is a centralized visual interface used to monitor and analyze key business metrics."

---
