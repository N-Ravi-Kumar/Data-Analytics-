# 📊 Spreadsheet Basics – Data Analytics Notes

## 📌 Overview

Spreadsheets are fundamental tools used in data analysis to organize, process, and visualize data in a tabular format. They are widely used due to their simplicity and accessibility.

---

## 🧩 1. Introduction to Spreadsheets

### 🔹 Key Concepts

* Used for storing, analyzing, and visualizing data
* Common tools: Excel, Excel Online
* Beginner-friendly and widely used in industry

### 💡 Example

* Analyzing monthly sales data using Excel

### 🎯 Interview Points

* Easy to use and widely available
* Suitable for small to medium datasets

---

## 📁 2. Spreadsheet Structure

### 🔹 Key Concepts

* **Workbook** → Entire file
* **Worksheet** → Individual sheet inside workbook
* File format: `.xlsx`

### 💡 Example

```
Workbook: Sales_Report.xlsx
   ├── January
   ├── February
   └── Summary
```

### 🎯 Interview Points

* Difference between Workbook and Worksheet
* XLSX is standard Excel format

---

## ⚙️ 3. Benefits of Spreadsheets

### 🔹 Key Concepts

* Data is organized in rows and columns
* Easy operations: Sorting, Filtering, Calculations
* Built-in features: Formulas, Pivot Tables, Charts

### 💡 Examples

* `=SUM(A1:A10)` → Calculate total sales
* Pivot Table → Sales by region

### 🎯 Interview Points

* One-stop solution for basic data analysis
* Commonly used in finance and reporting

---

## ⚠️ 4. Limitations of Spreadsheets

### 🔹 Key Concepts

* Difficult to track analysis steps
* Not suitable for large datasets
* Limited scalability

### 💡 Examples

* Complex formulas like `VLOOKUP` may break
* Handling millions of rows is difficult

### 🎯 Interview Points

* Not ideal for big data → use SQL/Python
* Poor reproducibility of analysis

---

## 🧠 5. Data Handling in Spreadsheets

### 🔹 Key Concepts

* Viewing data
* Entering data
* Editing data
* Navigating worksheets

### 💡 Examples

* Correcting wrong entries
* Moving across rows and columns

### 🎯 Interview Points

* Basic data cleaning and manipulation skills are essential

---

## 🚀 Final Summary

* Best for small to medium datasets
* Great for quick analysis
* Limited for advanced analytics

---

## 💥 Key Takeaways

* Workbook vs Worksheet
* Pivot Tables are powerful
* Easy but limited scalability
* Best starting tool for Data Analysts

---

## 🔗 Learning Path

```
Excel → SQL → Python → Advanced Analytics
```
